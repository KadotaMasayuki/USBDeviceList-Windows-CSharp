# USBDeviceList-Windows-CSharp
## Target
Windows10 ( or lower may be )
## Function
List bellow informations
* Device ID
* PNP Device ID
* Description
## Build
with cmd.com
> C:\Windows\Microsoft.NET\Framework\v3.5\csc.exe /platform:anycpu /target:exe USBDeviceList.cs
## Use
with cmd.com
> USBDeviceList.exe
